# react-redux-rollup-typescript-starter

This is a starter project utilizing following technologies:

  * React
  * Redux
  * Redux-Promise + Axios
  * TypeScript
  * Bootstrap/ Reactstrap
  * Rollup

## Usage

    # installation
    $ npm install

    # development mode
    $ npm run dev

    # production mode
    $ npm run build
    $ npm start

Open your browser at:

    http://localhost:3000/
